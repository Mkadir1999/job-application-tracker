# Job Application Tracker

Offline browser-based job application tracker built with HTML, CSS, and JavaScript.

Track applications, statuses, notes, and interview prep for IT Support, Help Desk, Desktop Support, and Junior Systems Administrator roles. All data stays in your browser via localStorage — no account or server required.

## Live demo

https://mkadir1999.github.io/job-application-tracker/

## Features

- Add and manage job applications with status tracking
- Store notes and interview prep per application
- Works fully offline after first load
- No backend or database required

## Tech

HTML · CSS · JavaScript · localStorage

## Run locally

Open `job-tracker/index.html` in a browser, or serve the `job-tracker` folder with any static file server.

## GitHub Pages setup

1. Repo **Settings → Pages → Build and deployment → Source:** GitHub Actions
2. Push to `main` — the workflow deploys the `job-tracker` folder automatically
